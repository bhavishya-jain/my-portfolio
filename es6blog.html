ES6 Features


1. Template Literals : We can now use back-ticks (`) for writing strings and to pass a value, we can use ${value}, for example : let name = `Your full name is ${firstName} ${lastName}`

2. Destructuring: It is a Javascript expression that can extract values from arrays or objects into a new variable. For example, const obj = { firstName: 'abc', lastName: 'xyz' }; const { firstName, lastName } = obj;

3. Arrow functions: The advantage of using arrow function is that, 'this' will have the same value as in the context of the function. For example, const a = () => console.log('Hello');

4. Promises: Promises provide us with asynchronous execution. For example, var asyncExample = new Promise((resolve, reject) => { // do something resolve(); }).then(()=> { console.log('Successful'); })

5. let and const: These allow us to scope the variable to a block. For example, function exampleFunc(bool) { let count= 0; if(bool) { let count = 1; // scope ends here for count } return count; } console.log(exampleFunc(true)); // output: 0
